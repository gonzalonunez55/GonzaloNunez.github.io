Descripción

Durante la asignatura Taller de Telemática del Tecnólogo en Telecomunicaciones, realicé un proyecto integrador que involucró el desarrollo de sistemas que combinan tecnologías de comunicaciones e informática. El proyecto consistió en diseñar y construir una solución que integra hardware y software, aplicando conocimientos de redes, servidores, bases de datos y programación de dispositivos.

Objetivos

  • Aplicar conocimientos de telecomunicaciones e informática en un proyecto real.
  • Integrar tecnologías de hardware y software para resolver un problema específico.
  • Aprender de manera autodidacta las herramientas y tecnologías necesarias para el proyecto.
  • Documentar y presentar de manera profesional los resultados del proyecto.

Metodología

  • Estudio guiado de temas relevantes al proyecto con apoyo del docente.
  • Desarrollo de un sistema de software y/o hardware funcional.
  • Investigación y aplicación de tecnologías de comunicación, redes, bases de datos y control de dispositivos.
  • Documentación técnica detallada y defensa oral del proyecto.

Tecnologías y habilidades aplicadas

  • Programación en C y Java para control de dispositivos y servidores.
  • Configuración de redes y protocolos TCP/IP para comunicación entre sistemas.
  • Diseño de bases de datos SQL para almacenamiento y consulta de información.
  • Desarrollo de aplicaciones web para interacción con hardware y bases de datos.
  • Gestión de dispositivos mediante interfaces serial, USB u otras.

Resultados y aprendizajes

  • Experiencia práctica en el desarrollo de sistemas telemáticos integrados.
  • Capacidad de aprendizaje autónomo para resolver problemas técnicos complejos.
  • Documentación profesional de un proyecto de ingeniería telemática.
  • Desarrollo de habilidades en programación, redes, bases de datos y control de hardware.
