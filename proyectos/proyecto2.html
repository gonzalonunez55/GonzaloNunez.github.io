Descripción

Durante la asignatura Taller de Telemática del Tecnólogo en Telecomunicaciones, desarrollé un proyecto integrador que combinó tecnologías de comunicaciones e informática. El objetivo fue diseñar y construir una solución que integra hardware y software, aplicando conocimientos de redes, servidores, bases de datos y programación de dispositivos. Para ello se utilizaron herramientas como routers TP-Link, celulares Android y un entorno virtualizado donde se implementaron diversos servicios de red.

Objetivos Realizados

  • Aplicar conocimientos de telecomunicaciones e informática en un proyecto real.
  • Integrar tecnologías de hardware y software para resolver un problema específico.
  • Implementar y administrar servicios de red, almacenamiento, seguridad y monitoreo.
  • Documentar y presentar de manera profesional los resultados del proyecto.

Metodología

  • Diseño y desarrollo de un sistema de software y hardware funcional.
  • Uso de entornos virtualizados para desplegar servicios críticos de red.
  • Investigación, configuración y aplicación de tecnologías de comunicación, servidores, bases de datos y aplicaciones móviles.
  • Documentación técnica detallada y defensa oral del proyecto.

Tecnologías y herramientas aplicadas

  • Routers TP-Link para la gestión y ruteo de la red.
  • Celulares Android como dispositivos clientes para pruebas.
  • DNS con Bind9 para resolución de nombres.
  • Servidor web Apache para servicios web.
  • Monitoreo con Zabbix para supervisión del estado de la red.
  • Wiki con Wiki.js como repositorio de documentación.
  • Nextcloud para almacenamiento y sincronización de archivos.
  • Base de datos PostgreSQL para almacenamiento estructurado.
  • Servidor VPN con OpenVPN para acceso remoto seguro.
  • Un Servidor central encargado de integrar la lógica de negocio, conectando la aplicación web y la aplicación Android con los datos recopilados de los Routers.
  • Programación en C, PHP y Java para control de dispositivos y servidores.
  • Configuración de redes y protocolos TCP/IP para comunicación entre sistemas.

Resultados y aprendizajes

  • Experiencia práctica en el desarrollo de sistemas telemáticos integrados.
  • Capacidad para implementar y administrar múltiples servicios en un entorno virtualizado.
  • Desarrollo de aplicaciones móviles y web conectadas a bases de datos y servidores.
  • Uso de herramientas de monitoreo, almacenamiento y colaboración en red.
